# Craft Cross CMS

[![License](https://img.shields.io/badge/license-Apache%202-blue)](https://github.com/plaidev/craft-cross-cms/blob/main/LICENSE)

A monorepo for all NPM packages related to Craft Cross CMS (**xcms**).

> **Note:** All packages are published under the `@craft-cross-cms` organization.

## Packages

### [@craft-cross-cms/rich-text-core](packages/rich-text-core)

TipTap-based rich text core library that provides HTML to JSON conversion functionality for rich text content.

## Getting Help

- **Have a bug to report?**
  [Open a GitHub issue](https://github.com/plaidev/craft-cross-cms/issues/new). If possible, include the library version and full logs.
- **Have a feature request?**
  [Open a GitHub issue](https://github.com/plaidev/craft-cross-cms/issues/new). Tell us what the feature should do and why you want the feature.
