# Craft Cross CMS Rich Text

[![License](https://img.shields.io/badge/license-Apache%202-blue)](https://github.com/plaidev/xcms-rich-text/blob/main/LICENSE)

A monorepo for all NPM packages related to Rich Text functionality in Craft Cross CMS.
This repository provides TipTap-based rich text editing and processing capabilities.

## Packages

### [@craft-cross-cms/rich-text-core](packages/core)

TipTap-based rich text core library that provides HTML to JSON conversion functionality for rich text content.

## Getting Help
